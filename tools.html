Sionna Munich AI-RAN

Sionna RT digital twin for AI-RAN: place gNodeBs in a Munich scene, solve planar radio maps on GPU, export SINR/throughput/outage KPIs, run optional finite-difference + Adam azimuth optimization, and ship a Word baseline-vs-optimized comparison..

Kernel Critic

An AI-driven orchestration framework designed to automate the generation and validation of high-performance CUDA kernels. Rather than relying on single-shot LLM prompts, KernelCritic implements a Reflective Agent architecture. It uses a "Critic" node to execute real-time nvcc compilations, feeding hardware constraint failures (e.g., memory coalescing, register pressure) back into the generation loop for automated self-healing.

Shapley value calculator

Shapley value tells us marginal contribution of a player in a cooperative game. During my master's thesis I created a tool to calculate Shapley value of a node in a nettwork. If we consider a graph with weighted nodes and edges, based on the definition of the payoff function, this simulator can calculate the Shapley value of each node of that graph for that payoff function.

Wireless link capacity calculator in urban areas

In semi or dense urban areas, usually there are mutiple floors in each building. We consider box type building distribution (usually found in large cities in the world like Manhattan, Barcelona) and we assume in each building there are multiple floors. During my master's thesis, I developed this tool using which wireless link capacity between any two floor can be calculated. Presenly, in this simulator, only a single street is considered. I have a plan for extending this simulator by considering multiple streets in a whole city.

Greedy graph partitioner

This tool is used to partition a garph following the greedy graph partitioning algorithm proposed in my master's thesis (Please check my publication list to get details about this algorithm). This algorithm partitions a graph into specified number of subgraphs where in each subgraph, all the nodes of that subgraph are within a specified number of hops away.
